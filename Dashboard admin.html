Reglas clave del dashboard
  • El dashboard muestra campañas paraguas, no sub-campañas. Cada row resume cuántas hijas tiene y en qué estado.
  • El avance de un paraguas es la suma de plantados / suma de metas de sus sub-campañas — no se edita aquí.
  • La actividad reciente lleva breadcrumb Campaña › Sub-campaña para no perder el origen del evento.
  • "Sin coordinador" en una row indica que al menos una sub-campaña no tiene responsable asignado.
  • El donut de estados sigue siendo del paraguas; los estados de sub-campaña se ven al entrar al detalle.
